## How to Use

**Download**

Download or clone this repo by using the link below:

```
https://github.com/abhishekpatelmc/hacksix.git
```

**Installation**

```
npm install
```

**To Start Server:**

```
npm start
```

**To Vist the App:**

```
http://localhost:3000
```
